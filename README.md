## datePicker(日期选择的组件)

1. 该组件基于html, css，和JavaScript原生实现的日期选择组件，更多的功能将在
未来添加

## 使用方法

> 1. clone  
   ```git clone git@github.com:sinianzhiren/datePicker.git```
> 2. 进入datePicker  
    ```cd datePicker```
> 3. 安装依赖  
    ```npm install / cnpm install```
> 4. 进入server
    ```cd server```
> 5. 安装依赖
    ```cnpm install / npm insatll```
> 6. 启动本地服务
    ```cd ../ && gulp --watch```
> 7. 打开浏览器访问localhost:3000/ 将看到日期组件的效果
